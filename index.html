🤖 안드로이드 (갤럭시 등)

크롬(Chrome)이나 삼성 인터넷 브라우저에서 아래 버튼을 눌러 바로 설치하세요.

설치 팝업이 뜨지 않으면 브라우저 메뉴(⋮)에서 '앱 설치'를 눌러주세요.

🍎 아이폰 (iOS)

반드시 Safari 브라우저로 열어주세요.
(카톡/네이버 인앱 브라우저 불가)

  1. 화면 하단 가운데 공유(📤) 버튼을 누릅니다.
  2. 메뉴를 내려 '홈 화면에 추가 ➕'를 누릅니다.
  3. 우측 상단의 '추가'를 누릅니다.
  4. 바탕화면에 생긴 앱 아이콘을 터치해 실행하세요!
