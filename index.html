당동 점수판 & 기록실

당동 앱은 일반 웹 브라우저에서는 실행되지 않습니다.
스마트폰에 전용 앱으로 설치한 뒤 빠르고 쾌적하게 즐겨보세요!

안드로이드 (갤럭시 등)

크롬(Chrome)이나 삼성 인터넷 브라우저에서 아래 버튼을 눌러 바로 설치하세요.

설치 팝업이 뜨지 않으면 브라우저 메뉴(⋮)에서 '앱 설치'를 눌러주세요.

아이폰 (iOS)

아이폰은 반드시 Safari(사파리) 브라우저로 설치해야 합니다.
(카톡/네이버 인앱 브라우저로는 설치되지 않아요.)

1️⃣ 카톡·네이버로 열었다면 → 먼저 사파리로 열기

  1. 화면 하단(또는 우측 하단 '···')의 공유 버튼을 누릅니다.
  2. 메뉴에서 'Safari로 열기'를 누릅니다.
    (이미 사파리라면 이 단계는 건너뛰세요.)

2️⃣ 사파리에서 앱 설치하기

  1. 화면 하단의 공유(⬆️) 버튼을 누릅니다.
  2. 메뉴를 내리거나 '더 보기 ∨'를 눌러 '홈 화면에 추가 ➕'를 누릅니다.
  3. 우측 상단의 '추가'를 누릅니다.
  4. 바탕화면에 생긴 앱 아이콘을 터치해 실행하세요!
